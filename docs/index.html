Frequently Asked Questions

Does this web launcher require game downloads? +
No! The web launcher streams and runs the game code directly inside Chrome's WebAssembly sandbox. You don't need to download or install the actual game on your laptop — everything runs in your browser tab.
How do I operate the virtual smartphone mod? +
Use the Up Arrow key to pull the phone up from the bottom of the screen. Navigate with Arrow Keys, press Enter to dial or trigger cheats, and Backspace / Escape to close the phone.
Is this project free and safe? +
Yes — 100% free and open-source. The launcher runs entirely locally inside your browser tab without sending any personal data. Licensed under the MIT license.
